Contracting

Silico Dynamics can be engaged on a contract basis to develop bioinformatics software, pipelines and platforms.

Software Development

Silico Dynamics can create custom bioinformatics software for your company, academic or government institution. Software is designed through cooperative assessment of the clients needs. Development follows an agile model in which the client is included every step of the way.

We are also able to build upon, adapt and improve open source software to meet the client's needs.

Pipeline Construction

Bioinformatics pipelines are often complex and troublesome to create. Existing open source tools are great on their own but are often troublesome to put together thus slowing pipeline development.

Silico Dynamics's developers can create these custom pipelines quickly and efficiently saving the client time and frustration.

Our pipelines are scalable, parallel and efficient allowing them to process large datasets (ie. "Big Data").

Cloud Service Creation

Silico Dynamics specializes in the creation of bioinformatics Software as Service (Saas) and Platform as a Service (Paas) systems. We can also develop various other web-based information systems such as Bioinformatics databases and repositories.

We can also create cluster or grid based bioinformatics analysis software which utilizes existing cloud infrastructure such as Amazon EC2.

Consulting

Silico Dynamics can assist clientele in the acquisition or development of bioinformatics software, pipelines and platforms.

Off-the-shelf Bioinformatics Software Assessment

There is an ever growing number of commercial and open source Off-the-shelf Bioinformatics software systems for clients to chose from and only a small subset of these meet their needs.

We can help clientele answer questions like the following:

  • Which software will scale with our data?
  • Which software will work with our current sequencing technology?
  • Which software will work with our future sequencing technology?
  • Which software will the most accurate answer to our biological questions?
  • What software can we use to replace our legacy systems?

Software Development Consultation

Due to the inherent complexity of processing and analyzing biological data many bioinformatics projects run into various road blocks during their development.

Silico Dynamics can evaluate these road blocks and suggest time efficient and cost-effective solutions.

We can also help clientele integrated the latest bioinformatic techniques and technologies into their pipelines and processes.

Our software development consultation services include:

  • Benchmarking and performance tuning of in-development or existing software.
  • Static code analysis and code quality assessment of in-development or existing software.
  • Recommendation of what new technologies should be integrated into existing software.
